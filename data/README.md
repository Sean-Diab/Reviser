# Data

This repository does not redistribute large raw datasets. Instead, it documents the public datasets and released prompt/evaluation manifests used in the paper.

Datasets referenced in the paper:
- C4 v2 English validation prompts for continuation benchmarks
- FineWeb-derived training data for Reviser and AR baselines

License and access notes:
- Users are responsible for complying with each dataset's license
- This repo only ships compact manifests and paper-facing summaries
- Large local caches, private preprocessing outputs, and raw checkpoints are intentionally excluded

See `data/manifests/` for released sample and artifact provenance.
