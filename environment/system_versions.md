# System Versions

Released environment targets:
- Python: 3.12
- PyTorch: CUDA-enabled build
- Transformers: 4.51.3
- CUDA runtime: 12.x or newer compatible build

Known experiment environment notes:
- CUDA GPUs were used for training and large-scale evaluation
- `transformers==4.51.3` is the pinned baseline for the released scripts
- Gated or third-party baselines may require separate credentials or licenses
